# chp1 연습문제

1.Q1
  - (a) 디바이스의 센서와 액추에이터의 역할을 비교하여 설명하세요.
    - 서로 반대되는 역할이다 센서는 입력 엑추에이터는 출력 이라고 할 수 있다, 센서는 주변 환경의 정보를 감지해서 전기 신호로 바꾸는 장치이고 엑츄에이터는 전기 신호를 받아서 실제 물리적인 동작을 수행하는 장치라고 정의할 수 있다
  - (b) 디바이스와 대시보드에서 동일한 프로그래밍 언어를 사용하지 않는 이유를 설명하세요.
    - 요구되는 기능, 실행 환경, 요구 사항 등 많은 것이 다르지만 크게 디바이스는 물리적인 것을 다루는 역할을 하고 대시보드는 프로그램 속에서 추상화된 것을 다루는 역할을 해서 각각에 맞는 프로그래밍 언어를 쓴다고 할 수 있다.
  - (c) 브로커와 같은 정보 전달 전담 서버의 이점을 설명하세요.
    - 정보전담 서버는 디바이스와 대시보드사이의 다리 역할을 하며 여러 이점이 있다, 많은 이점이 있지만 대표적으로 안정성과 확장성이 있다 중간 다리 역할이기에 한쪽이 변경되어도 다른쪽에 영향이 적고 직접 연결되지 않기 떄문에 트레픽관 관리도 용이 하다 그래서 안정성이 높고 브로커에서 다양한 메세지 프로토콜을 지원하고 다수에게 뿌리기 용이 하기에 확장성이 좋다고 할 수 있다. 

2. Q2
 -(c) mosquitto와 Node-RED에 대해 설명하고, 실행에 적합한 컴퓨팅 환결을 제시하십시오
  -mosquitto는 MQTT 프로토콜을 사용하는 메시지 브로커로 디바이스와 서버 사이에서 메시지 전달(중계) 역할을 수행 합니다, Node-RED는 Flow 기반 프로그래밍 도구로 IoT 데이터를 쉽게 처리하고 시각화 가능합니다
  -실행에 적합한 컴퓨팅 환경으로는 라즈베리파이, 개인 컴퓨터 등이 있고 라즈베리 파이가 둘다 설치 가능하며 다양한 프로젝트를 할수 있어서 추천됩니다
